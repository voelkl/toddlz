# Animal Playground

A colorful, interactive web game for toddlers. Tap animals to hear their sounds and watch them bounce!

## How to Play

1. Open `index.html` in any web browser
2. Tap or click on any animal
3. Watch it bounce and hear its sound!

## Features

- 8 colorful animals: Cow, Dog, Cat, Duck, Pig, Sheep, Frog, Bird
- Real animal sounds
- Bouncy animations on tap
- Shows animal names when tapped
- Touch-friendly with large tap targets
- Works on phones, tablets, and computers
- Parent controls (mute & fullscreen) in top-right corner
